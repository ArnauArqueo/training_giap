# Exercise: Introduction to historical morphology analysis
<img width="1230" height="840" alt="image" src="https://github.com/user-attachments/assets/2131a8c5-c337-4fc3-a86e-eccab62885b3" />

This Exercise is designed as an introduction to the analysis of historical morphology in the context of landscape archaeology. For this we will introduce the treatment and analysis of historical cartography and old aerial images in a GIS environment.    
 
The exercise has been created in a way that can be completed without any previous knowledge of GIS. It is recommended, however, to be familiar with the basic QGIS functions addressed in [e1](https://github.com/ArnauArqueo/training_giap/tree/main/e1_optical_telegraph) and [e2](https://github.com/ArnauArqueo/training_giap/tree/main/e2_photointerpretation_pyrenees). We will be using QGIS (the present document has been elaborated using the 3.44 version, note that some inconsistencies might arise due further updates. Contact your instructor if you encounter any difficulty).

To do the exercise download the [zip file](e4_archaeomorphology_emporda/Exercise_4_files.zip)  and follow the instructions from the [tutorial document](https://github.com/ArnauArqueo/training_giap/blob/main/e4_archaeomorphology_emporda/archaeomorphology_tutorial.pdf).
